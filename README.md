## What This Does

Every day `rustbot` creates a PR with:
- **3 MCQ questions** - Test your understanding
- **Learning material** - Concise tutorial on the topic
- **Coding problem** - 30-45 min exercise with tests

You solve directly in the PR, commit your answers, and merge when done.
